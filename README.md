# Nathanhruby Tap

## How do I install these formulae?

`brew install nathanhruby/tap/<formula>`

Or `brew tap nathanhruby/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "nathanhruby/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
